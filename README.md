# Campus Treats - BAUST Cafe Management System

Campus Treats is a modern Android application designed for BAUST students to browse, order, and manage cafe meals seamlessly. It features a secure login system and a dedicated admin portal.

## 🚀 Features
- **Student Authentication**: Secure login and registration.
- **Advanced Recovery**: Password recovery via Phone Number and Email using Firebase.
- **Digital Menu**: Browse categories and popular food items with real-time updates.
- **Cart & Ordering**: Add items to cart and track order history.
- **Admin Portal**: Dedicated dashboard for cafe owners to manage the menu and view orders.
- **Google Sign-In**: Quick access using Google accounts.

## 🛠 Tech Stack
- **Language**: Kotlin
- **UI Framework**: XML / Material Components
- **Backend**: Firebase (Authentication, Realtime Database, Storage)
- **Image Loading**: Glide
- **Design Style**: Theme-consistent Olive Green aesthetic

## ⚙️ Setup
1. Clone this repository.
2. Add your `google-services.json` to the `app/` directory.
3. Update Firebase Database rules to allow account recovery.
4. Build and run on an Android device (API 24+).
